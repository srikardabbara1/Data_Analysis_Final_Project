# Data_Analysis_Final_Project

Proposal: Sports bettors often assume favorites are safer bets, but sportsbooks price that expectation into the betting line. I want to analyze whether betting on NFL underdogs has historically produced a positive return on investment, and identify the key factors that make underdogs more likely to cover the spread or win outright. This would help answer whether there are systematic biases in NFL betting markets and whether certain game conditions make underdog bets more valuable.

Execution Plan:

By Week 5 Homework Deadline: Build a cleaned dataset where each row is one NFL game and includes: Favorite and underdog Point spread Moneyline odds Final score Home/away designation Team records entering the game Offensive and defensive stats entering the game Compute a baseline ROI analysis: Betting every underdog against the spread Potentially betting every underdog moneyline Define the main outcomes: Did the underdog cover? Did the underdog win outright?

By Week 7 Homework Deadline: Expand the dataset with additional predictive features such as: Home vs away underdog Divisional game or not Rest advantage/disadvantage Recent team performance Turnover differential entering the game Offensive/defensive efficiency measures Size of point spread Perform exploratory data analysis: ROI by spread bucket ROI by home underdog vs away underdog Win rate of underdogs by implied odds Build a model to identify the strongest factors associated with underdogs covering or winning: Logistic regression, decision tree, or random forest Visualize top predictive variables and compare model performance to the baseline strategy

By the final due date: Finalize analysis and present: Overall historical ROI of betting underdogs Whether any subgroups of underdogs are more profitable Which features most strongly predict underdog success Produce final deliverables: Clean notebook or report Visualizations of ROI and feature importance Clear conclusion on whether blindly betting underdogs is profitable and when underdog bets are most attractive
